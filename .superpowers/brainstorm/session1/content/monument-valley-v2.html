INTERACTION FLOW / 推演流程

视觉升级点（相对上一版）

  • 柔和色板：从黑金虚空改为纪念碑谷标志性的暖沙、薄荷、粉珊、靛蓝，整体明亮克制。
  • 塔楼建筑：每个 Agent 对应一座等距塔楼，带尖顶、拱门、小旗，不再是孤立方块。
  • 云雾背景：多层柔和云雾 + 远山剪影，增强浮空感与空间深度。
  • 小人物剪影：白袍小人在主平台行走，参考艾达的标志性造型。
  • Bridges 连接：金色实线 + 虚线叠加，表示已揭示的路径；远处仍有淡色阶梯。
  • 对话气泡：Agent 发言时塔楼旁浮现圆角气泡，文字为具体建议。
  • 装饰纹样：主平台中心有几何花纹，塔楼有拱门和尖顶细节，克制不花哨。

推演流程

  • 立卦：玩家输入决策问题
  • 选贤：演 Agent 根据问题标签选择 2-6 位 Agent 参与
  • 问答：Agent 轮流追问玩家，玩家回答
  • 聚议：所有 Agent 发言完毕，对应塔楼从云雾中升起
  • 定夺：演汇总生成 2-4 条路径 Bridge，玩家选择其一
  • 角色移动到选中塔楼，新的云雾升起，循环
