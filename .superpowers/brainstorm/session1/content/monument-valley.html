设计要点：

1. 等距视角（isometric）：所有方块用 30° 等距投影，三个面（顶/左/右）形成立体感，纪念碑谷核心视觉语言。

2. 虚空升起：方块从下方虚空（深色径向渐变）中升起，半透明 → 实体，伴随轻微上移 + 淡入。

3. Agent 对应方块：六位 Agent 各有专属色（钱谷金 / 路向绿 / 风眼红 / 心禾粉紫 / 镜渊紫 / 云图蓝），发言时对应方块升起。

4. 路径连接：方块升起后，金色细线连接成路径，形成 2-4 条可选出路。

5. 走过的淡去：已访问的方块降低不透明度（0.4），前方新一轮 Agent 发言触发新方块升起，循环。

6. 克制美学：底色纯黑虚空，方块米白 + 暖灰 + 沉棕，仅 Agent 色和沉金线作为强调，不花哨。
