## 未修复的已知问题

### P1: Three.js 资源未 dispose

- 位置: 全项目(LightOrb/NodesLayer/AgentGhosts 等约 15 处)
- 问题: `useMemo` 创建的 CanvasTexture/Geometry/Material 无 `useEffect` cleanup dispose
- 影响: 3D 场景反复挂载/卸载会持续泄漏 GPU 显存
- 评估: 当前单页应用,路由切换时旧页面卸载,影响有限。长期使用可能卡顿
- 建议: 在各 3D 组件添加 `useEffect(() => () => { tex.dispose() }, [tex])`

### P1: setTimeout 无 cleanup

- 位置: `ChatOverlay.jsx:90-138`、`ChatPanel.jsx:135-181`、`AgentPanel.jsx:68-85`
- 问题: useEffect 内 setTimeout 无 clearTimeout,组件卸载后仍 setState
- 评估: 这些组件当前未在主链路使用(ChatPanel/ChatOverlay 是旧版 UI),影响低
- 建议: 加 `return () => clearTimeout(id)`

### P2: 动态列表用 index 作 key

- 位置: `Game.jsx:401`、`ChatPanel.jsx`、`ChatOverlay.jsx`
- 评估: 消息列表动态增删,index key 可能导致 React diff 错误
- 建议: 改用唯一 id(如 `msg.id` 或时间戳)

### P2: 死代码

- `TravelerCharacter.jsx` 引用未定义的 `COLORS.traveler`,但未被任何文件 import
- `UserOrb.jsx` import 未使用的 COLORS
- 评估: 不会触发,但增加维护负担
- 建议: 删除或修复
