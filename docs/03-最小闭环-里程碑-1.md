**方式A：Vercel（推荐）**
1. 双击运行 `deploy-backend.bat`，选择1
2. 浏览器中完成Vercel登录
3. 等待部署完成，获得后端URL
4. 双击运行 `update-frontend.bat`，输入后端URL

**方式B：本地运行（开发测试）**
1. 在终端执行 `cd server && npm install && npm start`
2. 前端自动连接 localhost:3001
