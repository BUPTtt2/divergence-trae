## 二、数据分析示例

### 2.1 场景推荐分析

**场景**：家庭安全场景

**分析过程**：
1. 识别用户需求关键词："安全"、"家庭"、"老人"、"小孩"
2. 从产品知识库中检索相关产品：智能门锁、智能摄像机、智能门铃
3. 根据价格区间筛选：预算8000元内
4. 生成产品组合方案

**推荐方案**：
```
方案A（安全守护套装）：
- 小米智能门锁M4：2999元
- 小米智能摄像机3云台版：499元
- 小米手环8（老人版）：299元
- 小米智能门铃：599元
合计：4396元

方案B（全面安防套装）：
- 小米智能门锁M4：2999元
- 小米智能摄像机3云台版×2：998元
- 小米手环8×2：598元
- 小米Sound Pro（紧急呼叫）：599元
合计：5194元
```

### 2.2 议价策略分析

**用户输入**："太贵了，能不能便宜点？"

**分析过程**：
1. 识别议价意图：价格协商
2. 查询用户议价能力评分：中
3. 查询可用促销规则：满减、套装优惠、会员优惠
4. 计算可优惠空间

**议价方案**：
```
当前订单金额：8096元
可用优惠：
1. 满减优惠：满5000减500 → 优惠500元
2. 套装优惠：3件以上9折 → 优惠810元
3. 会员优惠：黄金会员95折 → 优惠405元

最优组合：
- 满减优惠 + 套装优惠 → 共优惠1310元
- 调整方案：门锁换成M3版，省500元
- 赠品：智能插座×3 + APP会员×1年（价值299元）

最终方案：7596元
```

## 四、数据可视化建议

### 4.1 仪表盘设计

**产品销售仪表盘**：
- 销售趋势图（折线图）
- 产品分类占比（饼图）
- 地区销售分布（地图）
- 实时库存预警（表格）

**用户画像仪表盘**：
- 用户分布热力图
- 消费能力分析（直方图）
- 活跃度趋势（折线图）
- 用户留存分析（漏斗图）

**促销效果仪表盘**：
- 促销ROI对比（柱状图）
- 转化率分析（漏斗图）
- 客单价变化（折线图）
- 优惠使用情况（饼图）

### 4.2 数据指标体系

| 指标类别 | 核心指标 | 计算方式 |
|----------|----------|----------|
| 销售指标 | 销售额 | 单价×销量 |
| 销售指标 | 销量 | 销售数量 |
| 销售指标 | 客单价 | 销售额/订单数 |
| 用户指标 | 活跃用户数 | 登录用户数 |
| 用户指标 | 用户留存率 | 留存用户/新增用户 |
| 用户指标 | 用户增长率 | 新增用户/总用户 |
| 促销指标 | 转化率 | 购买用户/访问用户 |
| 促销指标 | ROI | 销售额/促销成本 |
| 促销指标 | 优惠使用率 | 使用优惠订单/总订单 |
