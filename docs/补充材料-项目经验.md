# 相关项目经验

## 一、演策 · 3D罗盘起卦推演——当周易遇见AI

### 1.1 项目概述

**项目类型**：AI Agent决策辅助系统

**项目链接**：https://forum.trae.cn/t/topic/96005?u=yegua

**项目简介**：基于AI的多智能体决策辅助系统，将传统周易文化与现代AI技术相结合，通过3D罗盘交互界面，为用户提供智能决策推演服务。

### 1.2 核心功能

| 功能模块 | 描述 | 技术实现 |
|----------|------|----------|
| 多智能体协作 | 多个Agent围绕决策问题展开讨论，提供多角度分析 | LLM + Agent调度框架 |
| 对话式交互 | 用户与Agent进行自然语言对话，获取决策建议 | 对话管理系统 |
| 决策分支生成 | 根据用户选择生成不同的世界线，展示决策路径 | 状态机 + 分支逻辑 |
| 世界线区分 | 明确区分当前线、初始线和走过的线，通过颜色区分 | 前端可视化 |
| 智囊社区 | 用户可创建、分享、使用不同的智囊Agent | 社区系统 |
| 3D罗盘界面 | 水墨风格的3D交互界面，展示卦象推演过程 | Three.js + Canvas |

### 1.3 技术亮点

- **多智能体架构**：实现了多个Agent的协同工作，每个Agent有独立的角色定位和专业知识
- **动态分支系统**：支持决策路径的动态生成和可视化展示
- **对话管理**：成熟的对话状态管理，支持上下文理解和多轮对话
- **3D可视化**：水墨风格的3D罗盘，提供沉浸式交互体验

### 1.4 项目成果

- 已上线可演示，获得TRA E创造力大赛奖项
- 用户反馈良好，交互体验独特
- 技术架构完整，可扩展至其他领域

## 二、LearnForge · 可视化知识图谱

### 2.1 项目概述

**项目类型**：AI教育辅助系统

**项目链接**：https://forum.trae.cn/t/topic/52664?u=yegua

**项目简介**：提供1v1结构化讲解和个性化学习路径，通过知识图谱可视化，帮助用户高效学习。

### 2.2 核心功能

| 功能模块 | 描述 | 技术实现 |
|----------|------|----------|
| 知识图谱可视化 | 将知识结构以图谱形式展示，直观呈现知识点关系 | 图可视化引擎 |
| 智能讲解 | 根据用户学习状态，提供个性化的知识讲解 | LLM + 知识检索 |
| 学习路径规划 | 根据用户目标和水平，生成最优学习路径 | 路径规划算法 |
| 学习进度追踪 | 实时追踪学习进度，提供反馈和建议 | 数据统计 + 可视化 |
| 智能推荐 | 根据学习历史，推荐相关知识点和资源 | 推荐算法 |

### 2.3 技术亮点

- **知识图谱构建**：从非结构化文本中提取知识，构建结构化图谱
- **个性化推荐**：基于用户画像和学习历史的精准推荐
- **智能答疑**：基于RAG的智能问答系统
- **可视化呈现**：清晰的知识结构可视化

### 2.4 项目成果

- TRAE创造力大赛获奖项目
- 知识图谱构建技术成熟
- 个性化学习路径算法有效
